Home Automation with ESP32, DHT11, Servo, and Ultrasonic Sensor

This project simulates a home automation system where a servo motor acts as a door, controlled by ultrasonic sensor readings. The ESP32 microcontroller integrates with a DHT11 sensor to monitor environmental temperature and humidity. Based on the distance measured by the ultrasonic sensor, the servo motor opens or closes the door, providing an automated solution for managing access and environmental conditions in a smart home setup.

Reading Data from DHT11 and Displaying in Node-RED Dashboard

This project reads temperature and humidity data from a DHT11 sensor using an ESP32 and sends the values to the serial port. The data is then processed in Node-RED and displayed in a custom dashboard. This setup allows real-time monitoring of environmental conditions, with the data presented visually in charts and gauges, offering an intuitive IoT-based solution for tracking temperature and humidity.
